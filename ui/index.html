未找到Cursor数据库

系统未找到Cursor数据库文件，可能的原因：

  1. Cursor尚未运行过或未完成初始化
  2. Cursor数据库位于其他盘符的AppData文件夹中
  3. 使用了非标准安装路径
解决方案：
  1. 确保Cursor已正常运行过至少一次
  2. 或点击下方"手动搜索"按钮指定搜索路径
🌐
请关闭网络代理

检测到API请求失败，通常是由网络代理引起的，请按以下步骤操作：

  1. 关闭魔法或代理软件
  2. 断开代理连接（如有使用）
  3. 重新启动续杯助手
  4. 重新点击"刷新Cursor"按钮
提示：代理或魔法可能会阻止程序正常访问续杯服务器
